# EdgeX PC방 제안서

EdgeX 분산형 AI 인프라 구축을 위한 Web3 기반 엣지 컴퓨팅 솔루션을 PC방에 도입하기 위한 제안서 웹사이트입니다.

## 소개

이 프로젝트는 EdgeX 시스템을 PC방에 도입했을 때의 이점과 기대 효과를 설명하는 단일 페이지 웹사이트입니다.

## 기능

- EdgeX 시스템 개요 및 구성 요소 소개
- 도입 효과 시각화 (흐름도)
- PC방 인프라 적합성 분석
- PC방 점주 보상 구조 설명

## 기술 스택

- HTML5
- CSS3
- JavaScript (최소한)
- SVG 기반 다이어그램

```bash
# 클론 후 GitHub Pages 배포 방법

```
